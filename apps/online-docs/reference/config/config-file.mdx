---
title: "Configuration File"
description: "Reference for ~/.craft-agent/config.json"
---

The main configuration file stores your authentication method, model preferences, and workspace connections.

## Location

```
~/.craft-agent/config.json
```

## Structure

```json
{
  "authType": "oauth_token",
  "model": "claude-sonnet-4-5-20250929",
  "workspaces": [
    {
      "id": "ws-abc123",
      "name": "Personal Notes",
      "rootPath": "/path/to/workspace"
    }
  ],
  "activeWorkspaceId": "ws-abc123"
}
```

## Fields

### authType

How you authenticate with Claude AI.

| Value | Description |
|-------|-------------|
| `api_key` | Using Anthropic API key directly |
| `oauth_token` | Using OAuth token (Claude Max subscription or Craft account) |

### model

The default Claude model to use.

| Value | Model |
|-------|-------|
| `claude-opus-4-5-20251101` | Claude Opus 4.5 |
| `claude-sonnet-4-5-20250929` | Claude Sonnet 4.5 |
| `claude-haiku-4-5-20251001` | Claude Haiku 4.5 |

### workspaces

Array of configured workspaces. Each workspace represents a directory or project context.

```json
{
  "id": "ws-abc123",
  "name": "Personal Notes",
  "rootPath": "/Users/alex/projects/notes"
}
```

| Field | Description |
|-------|-------------|
| `id` | Unique identifier for the workspace |
| `name` | Display name you assigned |
| `rootPath` | Filesystem path to the workspace directory |

### activeWorkspaceId

The ID of the currently active workspace, or `null` if no workspace is selected. This determines which workspace context is used when Craft Agents starts.

## Example Configuration

```json
{
  "authType": "oauth_token",
  "model": "claude-sonnet-4-5-20250929",
  "workspaces": [
    {
      "id": "ws-personal",
      "name": "Personal Notes",
      "rootPath": "/Users/alex/notes"
    },
    {
      "id": "ws-work",
      "name": "Work Projects",
      "rootPath": "/Users/alex/work"
    }
  ],
  "activeWorkspaceId": "ws-personal"
}
```

## Modifying Configuration

Most settings can be changed through the app:

| Setting | Command |
|---------|---------|
| Model | `/model` |
| Workspace | `/workspace` |

For advanced changes, edit the file directly while Craft Agents is not running.

<Warning>
Back up your config file before manual edits. Invalid JSON will prevent Craft Agents from starting.
</Warning>
