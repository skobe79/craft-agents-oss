# Craft Agents Documentation

Documentation for [Craft Agents](https://github.com/lukilabs/craft-terminal-agent) - an AI terminal assistant that connects to your Craft workspace.

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview documentation changes locally:

```bash
npm i -g mint
```

Run at the root of the repository:

```bash
mint dev
```

View your local preview at `http://localhost:3000`.

## Publishing

Changes pushed to the default branch are automatically deployed via the Mintlify GitHub app.
