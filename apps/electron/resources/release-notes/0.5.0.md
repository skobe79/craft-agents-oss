# v0.5.0 — Pi SDK Backend

---

## Features

- **Pi SDK backend** — New out-of-process agent backend using the Pi SDK, supporting OpenRouter, Groq, Mistral, xAI, Cerebras, Hugging Face, Azure, Amazon Bedrock, Vercel, Google AI Studio, and more through a unified interface (2ec6e950, d88a65ff)
- **GitHub Copilot OAuth** — Device code flow for GitHub Copilot subscriptions via Pi SDK, with automatic browser open and token refresh (34a69f10, affc7562)
- **ChatGPT Plus OAuth** — Use your ChatGPT Plus subscription with Craft Agents via Pi SDK (34a69f10)
- **API key provider onboarding** — New onboarding flow for Pi API key providers with provider preset selector, model tier selection (Best/Default/Cheap) for providers with many models, and comma-separated input for custom endpoints (d588c0de, 1ddfe96f)
- **Provider icons** — Show upstream provider logos and names in settings for Pi connections (4b6807b3)
- **Google AI Studio support** — Native Google Search grounding for Pi backend (cbab0f87)
- **MCP source tool proxying** — Pi backend can proxy MCP source tools to the agent (d88a65ff)

## Improvements

- **Session list refactoring** — Extracted entity-list primitives for reusable list components (1cf13a6d)
- **Web-fetch hardening** — Protection against SSRF, unbounded reads, and edge cases (3904a786)
- **Explore mode security** — Block `find -exec` and similar flags (57990dfe)

## Bug Fixes

- **Copilot OAuth browser open** — Browser now opens automatically during GitHub device code flow (affc7562)
- **Config validator** — Added `pi` and `pi_compat` to valid provider types so config watcher stops rejecting Pi connections (affc7562)
- **Model tier preservation** — User-selected tier models no longer overwritten by SDK defaults or async model refresh (1ddfe96f)
- **Pi session tool rebuild** — Rebuild tools in-place instead of disposing session (eed9a468)
- **MCP proxy tool names** — Use single underscore for proxy tool names (7f6dcf3e)

## Internal

- Removed Codex/Copilot standalone backends — consolidated on Pi SDK (64c1fc7c)
- Pi permission propagation through subprocess architecture (d88a65ff)
- OAuth token refresh for PiAgent ChatGPT Plus flow (34a69f10)
