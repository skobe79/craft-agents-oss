# v0.4.0 — LLM Connections, Codex Support, Workspace Defaults

Multi‑connection AI setup, Codex/OpenAI support, per‑workspace defaults, and documentation upgrades.

---

## 🔌 LLM Connections (Multiple Providers)

ARCH Agentz OS now supports multiple **LLM connections**:

- Add multiple providers (Anthropic, OpenRouter, Codex/OpenAI, etc.)
- Validate and manage each connection independently
- Sessions lock to a connection after the first message

## 🤖 Codex / OpenAI Support

Connect OpenAI via **Codex** using OAuth and run Codex‑powered sessions alongside Anthropic connections.

## 🧭 Default Connection per Workspace

Workspaces can now set their own **default LLM connection**, overriding the global default for new sessions in that workspace.

## 🎨 Workspace‑Specific Default Themes

Each workspace can set a **default theme** via `defaults.colorTheme`, making it easy to visually distinguish contexts.

## 📚 Documentation Updates

New and updated docs covering:

- LLM Connections (new reference page)
- Codex/OpenAI configuration
- Multiple connections + session locking
- Per‑workspace default connection + theme
